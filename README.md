# Titanic EDA — Exploratory Data Analysis

Exploratory data analysis on the Titanic dataset using Python and Pandas.

## 📊 Key Findings
- Overall survival rate: **38.38%**
- Female survival rate: **74.2%** vs Male: **18.9%**
- 1st class survival: **63%** vs 3rd class: **24.2%**
- Passengers travelling alone had **30.4%** survival vs **50.6%** with family
- Most expensive ticket: **£512.33** (Cardeza, Thomas Drake Martinez)

## 🔧 Techniques Used
- Missing value treatment (median imputation, mode imputation, column dropping)
- GroupBy aggregations and multi-level grouping
- Feature engineering (FamilySize, IsAlone)
- Boolean indexing and filtering

## 📁 Dataset
[Titanic Dataset](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv)
— 891 passengers, 12 features
